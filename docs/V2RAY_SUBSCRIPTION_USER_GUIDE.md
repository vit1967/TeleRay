# Инструкция по настройке подписок TeleRay

## Вход в панель подписок

**Настройки → Auto-Connect → Подписки** 

Здесь вы управляете тремя типами подписок:
- **V2Ray Бот** — получение ключей через Telegram-ботов
- **MTProxy Канал** — прокси из публичных каналов
- **MTProxy Бот** — прокси через MTProxy-ботов

---

## 1. Добавление подписки

### Шаг 1: Нажмите «+ Добавить подписку»

Выберите тип:
- **V2Ray Бот** → откроется панель настройки V2Ray Бота
- **MTProxy Канал** → откроется панель настройки MTProxy Канала
- **MTProxy Бот** → откроется панель настройки MTProxy Бота

### Шаг 2: Выберите источник (бота/канал)

После открытия панели:
- Если поле **пустое** → окно выбора откроется автоматически
- Если поле **заполнено** → нажмите на него для перевыбора

**Для ботов:** Откроется окно выбора диалогов — выберите бота из списка (например, `@hlvpnbot`)

**Для каналов:** Откроется окно выбора каналов — выберите канал с прокси-конфигами

---

## 2. Настройка V2Ray Бота (Панель V2Ray Бота)

### Поля панели:

| Поле | Описание |
|------|----------|
| **Бот** | Username бота (например, `@hlvpnbot`) |
| **Запрос 1-14** | Команды боту (например, `/us1`, `/getkey`) |
| **Получить ключ по запросу N** | Кнопка отправки команды |
| **key fr.N** | Полученный ключ (vless://...) — только чтение |

### Заполнение полей «Запрос N»:

#### Вариант А: Пустое поле (короткий тап)
1. Тапните на пустое поле → откроется чат с ботом
2. Найдите и скопируйте команду (например, `/us1`)
3. Нажмите «Назад» → команда вставится из буфера автоматически

#### Вариант Б: Заполненное поле
- **Короткий тап** → редактирование текста (клавиатура)
- **Долгий тап** → выделение/копирование/вставка (стандартное меню Android)

#### Вариант В: Поле заканчивается на «;»
- Откроется чат с ботом для добавления новых команд
- Новые команды добавляются через «;»: `/us1; /us2; /getkey`

### Получение ключа:

1. Заполните поле «Запрос N» командой
2. Нажмите **«Получить ключ по запросу N»**
3. Бот ответит ключом (vless://...)
4. Ключ сохранится в поле «key fr.N» и отправится в v2rayNG

**Контекстное меню (долгий тап на кнопке получения):**
- Запросить/Обновить
- Показать ключ
- Отправить → v2rayTg
- Удалить

### Добавление новых команд:

Нажмите **«+ Добавить команду»** (скрыто при 14 командах):
- Добавится новая пара полей (Запрос N + Получить ключ)
- Автоматически откроется чат для копирования команды

## 3. Настройка MTProxy Канала (Панель MTProxy Канала)

### Поля панели:

| Поле | Описание |
|------|----------|
| **Канал подписки** | Название канала (например, «MTProxy Channel») |
| **Прокси N** | URL прокси из канала (tg://proxy?... или https://t.me/proxy?...) |
| **Тест N** | Результат тестирования (пинг или «недоступен») |

### Выбор прокси из канала:

1. Тапните на пустое поле «Прокси N» → откроется канал
2. Найдите сообщение с MTProxy-ссылкой
3. Скопируйте текст сообщения
4. Нажмите «Назад» → URL извлечётся и сохранится автоматически
5. Тест запустится автоматически

**Форматы URL:**
- `tg://proxy?server=example.com&port=443&secret=...`
- `https://t.me/proxy?server=example.com&port=443&secret=...`

### Тестирование прокси:

- Тапните на поле «Тест N» → повторный тест
- **Зелёный** (пинг X мс) → прокси работает
- **Красный** («недоступен») → прокси не работает
- **Оранжевый** («тестирование...») → идёт тест

**Контекстное меню (долгий тап на «Прокси N»):**
- Удалить
- Удалить и выбрать другой

## 4. Настройка MTProxy Бота (Панель MTProxy Бота)

### Поля панели:

| Поле | Описание |
|------|----------|
| **Бот** | Username MTProxy-бота |
| **Запрос N** | Команда боту (например, `/proxy`, `/get`) |
| **Прокси N** | Полученный URL + результат теста |

### Получение прокси от бота:

1. Заполните поле «Запрос N» командой (как в разделе 2)
2. Тапните на поле «Прокси N» → команда отправится боту
3. Бот ответит прокси-ссылкой
4. Прокси автоматически протестируется и добавится в список

**Статусы прокси:**
- **Ожидание** → запрос отправлен, ждём ответ
- **Тестирование** → проверка пинга
- **Зелёный** (ping X ms) → прокси добавлен
- **Красный** («недоступен») → прокси не работает
- **Таймаут** → бот не ответил за 60 сек

**Контекстное меню (долгий тап на «Прокси N»):**
- Запросить/Обновить
- Показать URL
- Удалить

## 5. Управление подписками

### Перевыбрать бота/канал:
1. Долгий тап на поле «Бот» или «Канал»
2. Выберите «Перевыбрать»
3. Выберите новый источник

### Удалить подписку:
1. Долгий тап на поле «Бот» или «Канал»
2. Выберите «Удалить подписку»
3. Все команды и ключи будут очищены

### Общие настройки (в главной панели):

| Настройка | Описание |
|-----------|----------|
| **Мин. раб. ключей** | Минимальное количество рабочих ключей для авто-обновления |
| **Мин. раб. MTProxy** | Минимальное количество рабочих прокси |
| **Интервал тестов (мин)** | Как часто проверять ключи/прокси |
| **Auto upd. MTproxy** | Вкл/выкл авто-обновление MTProxy |
| **tst v2rayTg→лучш.кл.** | Тестировать все ключи и выбрать лучший |
| **SEND KEYS TO V2RAY** | Отправить ключи в v2rayNG |

## 6. Типовые сценарии

### Сценарий 1: Настройка V2Ray с нуля
1. Настройки → Auto-Connect → Подписки
2. «+ Добавить подписку» → V2Ray Бот
3. Выберите бота (например, `@hlvpnbot`)
4. Тапните на пустое «Запрос 1» → скопируйте команду из чата
5. Нажмите «Получить ключ по запросу 1»
6. Дождитесь ключа (vless://...)
7. Вернитесь → нажмите «SEND KEYS TO V2RAY»

### Сценарий 2: Добавление резервного прокси
1. Откройте настройку подписки
2. «+ Добавить команду»
3. Скопируйте резервную команду
4. Получите ключ → он сохранится в «key fr.2»
5. Ключ автоматически отправится в v2rayNG

### Сценарий 3: Замена нерабочего ключа
1. Долгий тап на «Получить ключ по запросу N»
2. Выберите «Удалить»
3. Снова нажмите «Получить ключ по запросу N»
4. Бот выдаст новый ключ

### Сценарий 4: Настройка MTProxy из канала
1. «+ Добавить подписку» → MTProxy Канал
2. Выберите канал с прокси
3. Тапните на пустое «Прокси 1» → откроется канал
4. Скопируйте сообщение с прокси-ссылкой
5. Вернитесь → тест запустится автоматически
6. Зелёный статус → прокси добавлен в список

---

## 7. Советы

✅ **Автозаполнение из буфера:** После возврата из чата текст вставляется автоматически

✅ **Мульти-команды:** Используйте «;» для добавления нескольких команд в одно поле

✅ **14 команд на подписку:** Максимум 14 пар (запрос + ключ) на одну подписку

✅ **До 5 подписок:** Можно добавить до 5 подписок каждого типа

✅ **Авто-открытие:** Если поле пустое, окно выбора откроется само при входе в панель

❌ **Не копируйте лишний текст:** Для прокси копируйте только сообщение с URL (без комментариев)

❌ **Не удаляйте бота:** После удаления бота ключи перестанут обновляться

---

## 8. Решение проблем

| Проблема | Решение |
|----------|---------|
| **Окно выбора не открывается** | Тапните на поле «Бот» или «Канал» вручную |
| **Текст не вставляется из буфера** | Скопируйте команду ещё раз в чате |
| **Бот не отвечает** | Проверьте, не заблокирован ли бот; нажмите «Обновить» ещё раз |
| **Прокси не тестируется** | Убедитесь, что URL полный (с server, port, secret) |
| **Дублирование строк** | Выйдите и зайдите в панель заново |
| **Ключ не отправляется в v2rayNG** | Нажмите «SEND KEYS TO V2RAY» вручную |
